This repository is a copy of my private repository.
Since my privary repository has code for algorithms, it would be better to be invisible for CS learners. I wish they get the concept of algorithms from my project, and then study themselves.

The goal of this project is to visualize the process of algorithms step by step.
I believe it is better to see how an algorithm works rather than just memorizing the logic. 

The website contains arrays, trees, sorting, and pattern matching,and graph algorithms.

Belonging to a category means it will be visualized in the form of the category.
For example, heap belongs to tree category. It is visualized in the form of tree, but heap is not a tree, but an array.
